# Bach Chorale Dictionary
Bach Chorale Dictionary is a web-based platform for studying, analyzing, and practicing J.S. Bachâ€™s four-part chorales.

This site provides structured access to chorale metadata, score rendering, audio playback, melodic search, cadence classification, soprano-phrase grouping, keyboard harmony practice, and ear-training exercises.
Chorale data is sourced from music21.corpus.chorales.

## Purpose
The goal of this project is to offer an environment for learning Bach chorales:

- To explore chorales with searchable metadata

- To study cadences and phrase patterns

- To practice ear training and keyboard harmony

The platform is designed for composers, musicians, and anyone studying tonal harmony.

## Tools
### 1. Home

An introduction page linking to all modules of the project.

### 2. Chorale Dictionary

Lists all chorales with their metadata(Riemenschneider number, BWV, title, key, meter).

Displays full scores using OSMD.

Includes audio playback for each chorale using Tone.js Sampler.

Allows melody-based search, using both relative and absolute matching.

### 3. Cadence Dictionary

Groups cadence excerpts by cadence type(authentic, plagal/half, deceptive, phrygian, other).

Each cadence card displays the score with metadata.

### 4. Phrase Dictionary

Groups phrases that share the same soprano pattern.

Shows phrase metadata, measures, and an example score.

Button toggles between phrase-only and full versions.

### 5. Chorale Ear Training

Displays blank versions of chorale scores.

Button toggles between answer and exercise versions.

The pickup beat is displayed in the detail box.

### 6. Figured Bass

Provides bass-only MusicXML for harmonic realization practice.

Toggle between Figured Bass and Full Chorale for checking answers.
